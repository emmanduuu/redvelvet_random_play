# Red Velvet Random Music Video Player Script
I made this script to play a randomized red velvet music video for my hyprland setup. 

This will only play red velvet music videos or else it won't play. (hopefully, try to break the match filter so that I can make the filter tighter).

## Requirements
Make sure to have these installed or else the script will not run
1. kitty
2. mpv
3. yt-dlp

This is only tested and run on Arch Linux running Hyprland

## Usage
Run the script:
```bash
./play_random.sh
```
